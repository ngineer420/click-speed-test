<article class="prose">
  <h1>Project: cpsboost.com</h1>

  A free click speed test that measures clicks per second (CPS)

  <h2>Stack</h2>

  <ul>
    <li>- HTML/CSS/JS: Vanilla, no framework or build step</li>
    <li>- DevOps: Static files on GitHub Pages, custom domain via Cloudflare DNS</li>
  </ul>

  <h2>Objective</h2>

  To give players a fast, no-friction way to measure clicking speed across
  5s/10s/30s/60s timed modes and a 100-click race mode, with live CPS readout,
  a skill rating, and a personal best that persists across visits.

  <h2>Benefits</h2>

  <ul>
    <li>- Runs entirely client-side; no click data is ever uploaded</li>
    <li>- Personal best and results are stored only in the browser via localStorage</li>
    <li>- Works identically on desktop and mobile via the Pointer Events API</li>
    <li>- No backend or database, so the whole site is free to host</li>
  </ul>

  <h2>Drawbacks</h2>

  <ul>
    <li>- No account or cross-device sync — personal bests live only in one browser</li>
  </ul>

  <h2>Easter Eggs</h2>

  A tiny rabbit sits tucked into the very bottom-right corner of the page,
  a nod to eRabb.it (erabb.it) — the site this one was built alongside.
